Parksy Melody

v1.1.0

YouTube 오디오 커터. 프리셋 구간(10s/20s/30s/1min/3min)으로 클립을 잘라 Telegram 브릿지로 전송한다.

Audio
↓ Download APK ⌥ Source
Phone-led. YouTube URL을 붙여넣고 구간 프리셋을 선택하면 오디오 클립을 즉시 추출해 Telegram 채널로 전송한다. Parksy Wavesy의 후속작으로 YouTube 파이프라인에 특화.
Features
✂️
프리셋 클립 커터
구간 프리셋 선택 → 즉시 추출. 초 단위 조절 불필요.
10s 20s 30s 1min 3min
📱
YouTube 공유 인텐트 수신
YouTube 앱에서 "공유 → Parksy Melody"로 즉시 URL 전달.
📡
Telegram 브릿지 전송
추출된 오디오 클립을 Telegram 봇 파이프라인으로 자동 전송. 채널 배포 즉시 완결.
