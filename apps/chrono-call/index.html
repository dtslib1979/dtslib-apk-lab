Parksy ChronoCall

v1.0.0

통화 녹음 STT 변환기. OpenAI Whisper로 통화 녹음을 타임스탬프 세그먼트 텍스트로 변환한다.

Utility
↓ Download APK ⌥ Source
Phone-led. 삼성 One UI 녹음 폴더를 자동 탐지한다. 통화 녹음 파일을 탭 한 번으로 Whisper API에 전송하고, 결과를 타임라인 세그먼트로 표시한다. Parksy Capture와 auto-share로 연동된다.
Features
🎙️
Whisper STT 파이프라인
FFmpeg 전처리(mono 16kHz 64kbps) → OpenAI Whisper API → 타임스탬프 세그먼트 텍스트.
📱
삼성 녹음폴더 자동 탐지
One UI 4+ 경로 자동 스캔. 파일 탭 → 즉시 STT 시작.
📝
세그먼트 뷰어 + 오디오 싱크
타임스탬프 세그먼트 탭 → 오디오 해당 위치로 seek. 마크다운 내보내기 지원.
🔗
Parksy Capture 연동
auto-share 토글 ON → STT 완료 즉시 Parksy Capture로 GitHub 아카이브.
