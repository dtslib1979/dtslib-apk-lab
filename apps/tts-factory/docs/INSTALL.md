# TTS Factory 설치 매뉴얼

> 안드로이드 전용 | 개인 사용 목적

---

## 1. APK 다운로드

GitHub Releases에서 최신 APK 다운로드:

**https://github.com/dtslib1979/dtslib-apk-lab/releases**

파일명: `tts-factory-v1.x.x.apk`

---

## 2. 설치 전 설정

### 출처를 알 수 없는 앱 허용

1. **설정** → **보안** (또는 **생체 인식 및 보안**)
2. **출처를 알 수 없는 앱 설치** 선택
3. 다운로드에 사용한 앱(Chrome, 파일 관리자 등) 허용

> 삼성 기기: 설정 → 생체 인식 및 보안 → 출처를 알 수 없는 앱 설치

---

## 3. APK 설치

1. 다운로드된 APK 파일 탭
2. **설치** 버튼 탭
3. 설치 완료 후 **열기**

---

## 4. 앱 초기 설정

### 서버 연결 확인

앱 실행 시 자동으로 서버 연결 테스트:
- ✅ 녹색: 서버 정상
- ❌ 빨간색: 서버 오프라인

### 인증 키 입력

설정 화면에서:
- **Server URL**: `https://tts-server-498465562814.asia-northeast3.run.app`
- **App Secret**: 관리자에게 문의

---

## 5. 문제 해결

### 설치 안됨
- 저장 공간 200MB 이상 확보
- 출처를 알 수 없는 앱 허용 확인

### 서버 연결 실패
- 인터넷 연결 확인
- App Secret 정확히 입력

---

## 시스템 요구사항

- Android 8.0 (API 26) 이상
- 저장 공간 200MB 이상
- 인터넷 연결 필수
