# TTS Factory 사용 매뉴얼

---

## 기본 사용법

### 1. 텍스트 입력

- 직접 입력: 텍스트 영역에 변환할 내용 입력
- 파일 불러오기: `.txt` 파일 선택

### 2. 음성 설정

| 프리셋 | 설명 |
|--------|------|
| neutral | 중립적 톤 (기본값) |
| news | 뉴스 앵커 스타일 |
| story | 나레이션 스타일 |

### 3. 언어 선택

| 코드 | 언어 |
|------|------|
| ko | 한국어 |
| en | 영어 |
| ja | 일본어 |
| zh | 중국어 |

### 4. 변환 실행

1. **변환** 버튼 탭
2. 진행률 확인 (0% → 100%)
3. 완료 후 **다운로드** 버튼 활성화

### 5. 결과 다운로드

- ZIP 파일로 다운로드
- 내부 구조: `{id}.mp3` 파일들

---

## 고급 기능

### 배치 처리

여러 텍스트를 한 번에 변환:

1. 구분자 `---`로 텍스트 분리
2. 각 섹션이 개별 MP3로 생성

예시:
```
첫 번째 문장입니다.
---
두 번째 문장입니다.
---
세 번째 문장입니다.
```

### 텍스트 제한

- 유닛당 최대: 1,100자
- 요청당 최대: 25개 유닛
- 초과 시 자동 분할

## 에러 메시지

| 메시지 | 원인 | 해결 |
|--------|------|------|
| Invalid secret | 인증 실패 | App Secret 확인 |
| Max 25 items | 유닛 초과 | 텍스트 분할 |
| Text too long | 글자 수 초과 | 1100자 이하로 수정 |
| Server error | 서버 오류 | 잠시 후 재시도 |

## 팁

- 긴 문장은 마침표 기준으로 분할하면 자연스러움
- 숫자는 한글로 풀어쓰면 발음 정확도 향상
- 영어 약어는 풀네임으로 작성 권장
