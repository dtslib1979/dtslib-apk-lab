Phone-led. 방송 중 사고 단계(Axis)를 화면 위에 실시간으로 오버레이한다. shareData IPC를 통해 같은 기기의 다른 Parksy 앱과 상태를 동기화한다. v11까지 누적된 방송 실사용 최적화.
Features
🧠
사고 단계 실시간 표시
방송/녹화 중 현재 사고 단계를 오버레이로 표시. 발표자 프롬프트 역할.
🔄
shareData IPC 동기화
같은 기기의 Parksy Studio, Parksy Glot 등과 실시간 상태 공유.
🎬
방송 파이프라인 통합
Parksy Studio와 함께 사용하면 방송 전체 상태를 한 화면에서 제어.
