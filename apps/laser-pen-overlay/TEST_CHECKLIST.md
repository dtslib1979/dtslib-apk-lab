## 🆕 v2.0 시스템 오버레이

### 오버레이 활성화

- [ ] 앱 실행 → "오버레이 OFF" 버튼 표시됨
- [ ] 버튼 탭 → "오버레이 ON" 전환
- [ ] 안내 화면 표시 ("오버레이 모드 활성화")
- [ ] 홈 버튼 → 홈 화면으로 이동

### 웹브라우저 위 판서

- [ ] Chrome 실행
- [ ] S Pen으로 화면에 판서 → **오버레이에 선 표시**
- [ ] 손가락으로 스크롤 → **웹페이지 스크롤됨**
- [ ] 손가락으로 링크 탭 → **링크 이동**
- [ ] 3초 대기 → 선 자동 소멸

### YouTube 위 판서

- [ ] YouTube 실행
- [ ] S Pen으로 판서 → 선 표시됨
- [ ] 손가락으로 스크롤 → 피드 스크롤됨
- [ ] 손가락으로 영상 탭 → 영상 재생

## 기본 기능 테스트

### 판서 (앱 내 모드)

- [ ] S Pen으로 선 그리기 → 선 표시됨
- [ ] 3초 대기 → 선 자동 소멸 (fade-out)
- [ ] 여러 선 그리기 → 각각 3초 후 소멸

### 색상 전환

- [ ] 🎨 버튼 탭 → 색상 변경 (흰→노→검→흰)
- [ ] 변경된 색상으로 그리기 → 해당 색상 표시

### Undo/Redo

- [ ] 선 그린 후 ◀ Undo → 마지막 선 제거
- [ ] ▶ Redo → 선 복원
- [ ] 여러 번 Undo → 순차적 제거

### Clear/Exit

- [ ] 🧹 Clear → 모든 선 제거
- [ ] 🚪 Exit → 앱 + 서비스 종료
