# Laser Pen Overlay 테스트 체크리스트

## 환경

- [ ] Galaxy Tab S9
- [ ] Android 14+
- [ ] S Pen 준비

## 설치 테스트

- [ ] APK 다운로드 (GitHub Actions Artifacts)
- [ ] APK 설치 성공
- [ ] 앱 실행
- [ ] 오버레이 권한 요청 화면 표시
- [ ] 권한 허용 후 판서 화면 진입

## 기본 기능 테스트

### 판서

- [ ] S Pen으로 선 그리기 → 선 표시됨
- [ ] 선 그린 후 3초 대기 → 선 자동 소멸 (fade-out)
- [ ] 여러 선 그리기 → 각각 3초 후 소멸

### 색상 전환

- [ ] 색상 버튼 탭 → 색상 변경 (흰→노→검→흰)
- [ ] 색상 버튼 스와이프 → 색상 변경
- [ ] 변경된 색상으로 그리기 → 해당 색상으로 표시

### Undo/Redo

- [ ] 선 그린 후 Undo → 마지막 선 제거
- [ ] Undo 후 Redo → 선 복원
- [ ] 여러 번 Undo → 순차적 제거

### Clear/Exit

- [ ] Clear 탭 → 모든 선 제거
- [ ] Exit 탭 → 앱 종료

## 터치 분리 테스트

### S Pen 전용 모드 (S Pen 배지)

- [ ] S Pen으로 그리기 → 선 표시됨
- [ ] 손가락으로 그리기 시도 → 선 안 그려짐 (하위 앱으로 전달)

### 모든 입력 모드 (All 배지)

- [ ] 우측 상단 배지 탭 → 모드 전환
- [ ] S Pen으로 그리기 → 선 표시됨
- [ ] 손가락으로 그리기 → 선 표시됨

## 하위 앱 연동 테스트

### 크롬 브라우저

- [ ] 크롬 실행 후 Laser Pen 실행
- [ ] S Pen으로 판서 → 판서 표시
- [ ] 손가락으로 스크롤 → 크롬 스크롤됨
- [ ] 손가락으로 링크 탭 → 링크 이동

### YouTube

- [ ] YouTube 실행 후 Laser Pen 실행
- [ ] S Pen으로 판서 → 판서 표시
- [ ] 손가락으로 스크롤 → YouTube 스크롤됨

## 화면녹화 테스트

- [ ] 화면녹화 시작
- [ ] S Pen 판서 → 녹화에 판서 표시됨
- [ ] 페이드아웃 → 녹화에도 페이드아웃 표시
- [ ] 컨트롤 바 → 녹화에 표시됨

## 성능 테스트

- [ ] 빠른 판서 시 렛 없음
- [ ] 여러 선 동시 페이드아웃 시 버벼거림 없음
- [ ] 장시간 사용 시 메모리 누수 없음
