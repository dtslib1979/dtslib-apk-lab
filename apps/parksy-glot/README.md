# Overlay Dual Subtitle APK

S Pen 웹 오버레이 판서 앱 - Galaxy Tab S9용

## Features (v1.0.0)

- S Pen으로 웹페이지 위에 판서
- 손가락으로 스크롤/터치 조작
- 레이저펜 효과 (3초 후 Fade-out)

## Tech Stack

- **Framework**: Flutter 3.24.0
- **Platform**: Android (Kotlin native bridge)
- **Target**: Galaxy Tab S9 (Android 13+)

## Build Status

This is a Flutter project. Build with:

```bash
flutter pub get
flutter build apk --debug
```
