실시간 다국어 자막 오버레이. OpenAI Whisper + GPT-4o로 음성을 받아 화면 위에 번역 자막을 즉시 표시한다.

Subtitle System Overlay
↓ Download APK ⌥ Source
Phone-led. 회의, 강의, 외국어 콘텐츠 시청 중 실시간으로 음성을 인식하고 번역 자막을 화면 위에 오버레이한다. Whisper로 전사하고 GPT-4o로 번역. 폰 하나로 동시통역.
Features
🎤
실시간 Whisper 전사
마이크 입력을 OpenAI Whisper로 실시간 전사. 다국어 자동 감지.
🔄
GPT-4o 즉시 번역
전사 텍스트를 GPT-4o로 즉시 번역. 한/영/일/중 등 주요 언어 지원.
📺
전체 화면 오버레이 자막
번역 결과를 어느 앱 위에서든 자막 형태로 오버레이. 불투명도/위치 조절 가능.
